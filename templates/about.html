Welcome!

The Exoplanet Classifier is a project developed by Team Ontohin 4b for the NASA Space Apps Challenge 2025.

Objective

The objective of this web app is to predict new exoplanet‚Äôs status (CONFIRMED, CANDIDATE, FALSE POSITIVE) by evaluating user-given inputs.

How does it work?

The user-given data is sent to the model via a Flask API endpoint. Then, the model predicts the classes and also returns a probability for each class (which you can see after hitting the ‚ÄúPredict‚Äù button).

We have used the widely known Scikit-learn library and its RandomForestClassifier to build it from the ground up. It is an ensemble model which works by creating a bunch of decision trees and training them with slightly different features. If you want to learn more about it, click here to see the official docs.

This is one of the most widely used models provided by Scikit-learn and also very suitable for our objective. It predicts the result in three different classes:

  • CONFIRMED (2) ‚Üí The disposition represents a confirmed exoplanet.
  • CANDIDATE (1) ‚Üí The disposition is neither a confirmed exoplanet nor a false positive. It requires further calculation to determine its actual status.
  • FALSE POSITIVE (0) ‚Üí The disposition does not represent an exoplanet.

The model also returns a probability (up to 3 decimal digits) of its prediction for each of those 3 classes.

Things to Keep in Mind

  • No ML model is perfectly accurate ‚Äî the same goes for this one.
  • We tried to use more than just the KOI dataset. We tested with more data from other open-source datasets provided by NASA and its partners. Unfortunately, those introduced more noise and caused overfitting.
  • Within the short time period, we could not come up with a better solution, so we continued with just one dataset.
